Realtime Motion Capture
Goal of this project is to capture motion data using motion capture suit And process the data in realtime to record the movements and feed the data to the character in the virtual world. This project was initiated on 24th August, 2021. And it was completed on 12th July, 2022. My team included four other members. This project was one of my huge success. Pathfinding AI using Neural Networks
This is one my of solo project. In this project, I worked in neural networks to make an AI agent that can find path in a dynamic environment. I had the chance to implement various concepts like forward propation and back propagation. It was a great learning experience for me and this was the first time I stepped into the world of neural networks to widen my knowledge in computer science.
